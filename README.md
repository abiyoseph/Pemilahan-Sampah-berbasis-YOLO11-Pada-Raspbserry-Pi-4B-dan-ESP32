# Smart Waste Sorting System Using YOLOv11 and Inductive Proximity Sensor

## Overview

This project implements an intelligent waste sorting system using:

- YOLOv11n
- Raspberry Pi 4B
- ESP32
- USB Camera
- Inductive Proximity Sensor

The system automatically classifies waste into:

- Plastic
- Paper
- Metal

and improves metal detection reliability using sensor fusion.

---

## Hardware

- Raspberry Pi 4B
- ESP32
- USB Camera
- Inductive Proximity Sensor
- Servo Motor

---

## Software

- Python
- Ultralytics YOLOv11
- OpenCV
- Arduino IDE

---

## Dataset

Custom dataset:

- Plastic

- Metal
